Códigos Úteis do Kali Linux

1. Atualizar os repositórios e pacotes

sudo apt update
sudo apt upgrade

2. Instalar uma ferramenta (exemplo: Nmap)

sudo apt install nmap

3. Verificar as interfaces de rede

ifconfig

4. Escanear portas em um host

nmap <IP_do_host>

5. Iniciar o Apache Web Server

sudo service apache2 start

6. Iniciar o PostgreSQL (para uso com o Metasploit)

sudo service postgresql start

7. Iniciar o Metasploit Framework

msfconsole

8. Encontrar redes sem fio disponíveis

iwlist wlan0 scan

9. Crackear senhas WPA/WPA2

airodump-ng -c <canal> --bssid <BSSID> -w <nome_do_arquivo> wlan0
    aircrack-ng -w <dicionário> -b <BSSID> <nome_do_arquivo>.cap
