Códigos Úteis do Windows

1. Verificar a versão do Windows

winver

2. Abrir o Gerenciador de Tarefas

Ctrl + Shift + Esc

3. Limpar a memória RAM

Memory Diagnostic

4. Verificar o espaço em disco

wmic logicaldisk get size,freespace,caption

5. Desligar o computador

shutdown /s /t 0

6. Reiniciar o computador

shutdown /r /t 0

7. Encerrar um processo pelo nome

taskkill /F /IM nome_do_processo.exe

8. Verificar informações do sistema

systeminfo

9. Criar um arquivo vazio

type nul > nome_do_arquivo.txt

10. Alterar a senha de um usuário

net user nome_do_usuário nova_senha
