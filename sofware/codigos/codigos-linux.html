Códigos Úteis para Usuários do Linux

1. Atualizar o sistema

sudo apt update
sudo apt upgrade

2. Listar arquivos e diretórios

ls

3. Criar um diretório

mkdir nome_do_diretório

4. Copiar um arquivo ou diretório

cp arquivo_origem arquivo_destino

5. Mover ou renomear um arquivo ou diretório

mv arquivo_origem arquivo_destino

6. Remover um arquivo ou diretório

rm arquivo_ou_diretório

7. Visualizar o conteúdo de um arquivo

cat nome_do_arquivo

8. Conectar-se a um servidor remoto via SSH

ssh nome_de_usuário@endereço_do_servidor
